We are using FastAPI for the backend, JSON Web Tokens (JWT) for authentication and PostgreSQL for our database. The frontend is built with React and designed with Bootstrap and CSS.

Technologies: React, JavaScript, FastAPI, Python3, PostgresSQL, Docker, Swagger, JSX, Bootstrap, HTML5, CSS
